========================================
   GUILD SHOP PRICE SYNC v2.6
   Thralls Book Club
========================================

WHAT THIS DOES:
---------------
Scans Auction House prices in-game and
automatically updates the guild website
so everyone sees current prices.


HOW TO USE (after installing):
------------------------------
1. Open World of Warcraft
2. Go to the Auction House
3. Click "Sync Prices" button (bottom-right)
4. Wait for scan to finish
5. Click "Reload UI & Sync"
6. Done! Prices update on website automatically


TO UNINSTALL:
-------------
Option 1: Windows Settings
  - Open Windows Settings
  - Go to Apps > Installed Apps
  - Find "GuildShopSync"
  - Click Uninstall

Option 2: Start Menu
  - Open Start Menu
  - Find "GuildShopSync" folder
  - Click "Uninstall GuildShopSync"
